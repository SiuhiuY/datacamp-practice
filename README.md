## Exercises

| Exercise | Description | Skills Practised |
|----------|-------------|------------------|
| **Building REST APIs** | Retrieve historical exchange rates from a public REST API and integrate them into international sales data for currency conversion. | REST API consumption, HTTP requests with requests, query parameters, JSON parsing, dictionary mapping, Pandas data manipulation
| **Functions to Register App Users** | Build a simple user registration workflow by validating user inputs and handling invalid data. | Functions, modular programming, input validation, exception handling, dictionaries, and clean code practices |
| **Getting a Good Night's Sleep** | Analyze sleep patterns across occupations and BMI categories to identify factors associated with sleep quality and insomnia. | Data cleaning, exploratory data analysis, descriptive statistics, grouping and aggregation, ratio calculation, and data visualization with Pandas and Matplotlib |
| **Investigating Netflix Movies** | Explore trends in movie duration over time using Netflix metadata. | Data manipulation, filtering, feature selection, exploratory data analysis, categorical encoding, and data visualization with Matplotlib |
| **Analyzing Students' Mental Health** | Analyze the relationship between international students' length of stay and mental health outcomes using SQL. | SQL querying, filtering, aggregation, GROUP BY, summary statistics, data analysis, and analytical reporting |
